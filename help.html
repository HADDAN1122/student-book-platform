1. Common Console Errors & Solutions

Error: "Firebase is not defined"
Cause: Firebase SDK not loaded
Fix: Add this before your script:
<script src="https://www.gstatic.com/firebasejs/8.10.0/firebase-app.js"></script>

Error: "auth is not defined"
Cause: auth.js not loaded or wrong order
Fix: Load files in correct order:
1. firebase.js
2. auth.js
3. your-script.js

Error: "db.collection is not a function"
Cause: Firestore not initialized
Fix: Initialize Firestore in firebase.js:
const db = firebase.firestore();

Error: "CORS policy blocked"
Cause: Wrong Firebase config or rules
Fix: Check Firebase console rules:
allow read, write: if true; (for testing)
